# UW Assessment Tool Mobile

This Expo React Native wrapper loads the full UW Assessment Tool HTML inside a WebView so the same functionality is available on mobile.

## Prerequisites

- Node.js 18+
- Expo CLI (`npm install -g expo-cli`)

## Install

```bash
cd mobile
npm install
```

## Run

```bash
npm run start
```

Then follow the Expo prompts to launch on iOS/Android simulators or devices.
